# notebooks/

**EDA only.** This directory is for exploratory data analysis.

Notebooks here are NOT the source of truth for any analytical result.
Canonical analyses live in `scripts/` (runnable entry-points) and
`workflow/rules/` (Snakemake rules).

Notebooks are stripped of output cells by `nbstripout` on commit.

Naming convention: `YYYY-MM-DD-<short-description>.ipynb`
